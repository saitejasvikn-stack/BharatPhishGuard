# 🛡️ BharatPhishGuard
**BharatPhishGuard** is an AI-driven cybersecurity tool designed to detect and prevent phishing attacks specifically targeting Indian users.

## 🚀 Features
* **Web Interface:** Built with Streamlit for easy URL/Content checking.
* **Telegram Integration:** A dedicated bot for real-time phishing alerts.
* **AI-Powered:** Uses Machine Learning and Google Gemini AI to analyze threats.
* **Database Logging:** Keeps track of reported threats using SQLite.

## 🛠️ Tech Stack
* **Language:** Python
* **Frontend:** Streamlit
* **AI/ML:** Scikit-learn, Google GenAI (Gemini)
* **API:** Telegram Bot API
